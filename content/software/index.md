**Cost-Sensitive Neighborhood Aggregation (CSNA) for Heterophilous Graphs** — The code for our new arXiv paper, about per-edge routing in Graph Neural Networks (GNNs), is available [here](https://github.com/CRL-Technion/CSNA-public).

**Multi-Agent Warehouse Rearrangement** — The code for our SoCS 2025 paper, about optimal multi-agent warehouse rearrangement, is available [here](https://github.com/CRL-Technion/wh-rearrangement).

**BLITstar** — The code for the ICRA 2025 paper "Asymptotically Optimal Sampling-Based Motion Planning Through Anytime Incremental Lazy Bidirectional Heuristic Search", about motion planning, is available [here](https://github.com/yi213-robotic/BLITstar-ICRA2025). It is also incorporated in [OMPL](https://ompl.kavrakilab.org/).

**MEET** — The code for our IJCAI 2025 paper, about bidirectional search, is available [here](https://github.com/yi213-robotic/MEET). It will also be incorporated in [OMPL](https://ompl.kavrakilab.org/).

**Numeric Fast Downward** — Our paper on numeric PDBs is implemented in the numeric planner [NFD](https://github.com/dgnad/numeric-fast-downward).

**PlanDEM** — Papers from my PhD were mostly tested using [PlanDEM](https://github.com/eyal-weiss/plandem-public), a domain-independent planner that provides implementation to algorithms that work with dynamically estimated action models. This is an open source project, that gets major updates typically after a new paper on the matter is published.
